LVS Barcode- & Etikettensystem

Dieses System wurde für den Hochgeschwindigkeits-Druck von Barcode-Etiketten in Lagerumgebungen entwickelt. Es arbeitet datenschutzkonform direkt im Browser-Speicher und benötigt keine permanente Datenhaltung für Ihre CSV-Listen.

1. Das mentale Modell: Projekt vs. Daten

Um das System effizient zu bedienen, müssen Sie den Unterschied zwischen einem Projekt und der CSV-Datei verstehen:

Das Projekt (Beständig)

Ein Projekt ist Ihre Druck-Schablone. Es speichert dauerhaft:

  • Physisches Maß: Breite/Höhe des Aufklebers (z.B. 100x50mm).
  • Papierart: A4-Bögen (z.B. 2x4 Etiketten) oder Endlos-Rolle.
  • Design: Wo stehen Textfelder, wo sind Barcodes?
  • Felderkennung: Welche Namen haben die Spalten Ihrer CSV-Datei (z.B. ArtNr).
Die CSV-Daten (Flüchtig)

Die Liste der echten Artikel, die Sie morgens aus dem ERP-System exportieren, ist flüchtig:

  • Sie werfen die Liste in das Projekt ("Reload CSV").
  • Sie drucken Ihre Etiketten.
  • Sobald Sie den Browser schließen, werden diese Artikeldaten automatisch gelöscht.

2. Der tägliche Workflow

Folgen Sie diesen Schritten im Arbeitsalltag:

1
Projekt wählen

Klicken Sie auf der Startseite auf das entsprechende Projekt (z.B. "Logistik-Standard").

2
Frische Daten laden

Nutzen Sie oben rechts den Button Reload csv, um Ihre aktuelle Export-Datei hochzuladen.

3
Suchen & Markieren

Nutzen Sie die Suchfelder über der Tabelle im Reiter "DATEN". Setzen Sie die Haken bei allen Zeilen, die auf den Drucker sollen.

4
Druckvorschau & PDF

Klicken Sie auf "Drucken". Passen Sie ggf. die Startposition an (bei Bögen) und klicken Sie auf "Jetzt Drucken (PDF)".

3. Der Visuelle Designer

Im Reiter "DESIGNER" gestalten Sie das Aussehen Ihres Etiketts.

Objekte hinzufügen & bearbeiten

  • Klicken Sie auf "+ Text" oder "+ Barcode", um ein neues Element auf dem Etikett zu platzieren.
  • Drag & Drop: Ziehen Sie das Objekt mit der Maus an die gewünschte Position.
  • Bearbeiten: Mit einem Doppelklick auf das Objekt (oder Klick auf das ✏️-Icon) öffnen sich die Eigenschaften.

Barcode-Typen

Sie können zwischen drei Haupttypen wählen:

Code 128
Standard-Code für Texte/Zahlen.
EAN 8 / 13
Europäische Artikelnummern (8/13 Ziffern).
QR-Code
Große Datenmengen / Weblinks.

Präzisions-Werkzeuge & Multi-Selektion

Multi-Selektion

Halte die Strg-Taste gedrückt, um mehrere Elemente auszuwählen. Diese können dann gemeinsam verschoben oder über die Toolbar ausgerichtet werden.

Ausrichtungs-Toolbar

Nutze die Icons im Kopfbereich zum linksbündigen Ausrichten, Angleichen der Breite oder zum gleichmäßigen vertikalen Verteilen von Elementen.

EAN Validierung

EAN8 und EAN13 Barcodes werden in Echtzeit geprüft. Enthalten diese nicht die benötigte Anzahl an Ziffern (8 bzw. 12-13), erscheint eine Warnmeldung im Designer und ein deutlich sichtbarer "ERROR" auf dem Ausdruck.

Profi-Tipp: QR-Code Format

Das System achtet beim QR-Code automatisch auf ein quadratisches Format (1:1). Ändern Sie die Breite, zieht die Höhe im Designer automatisch mit, um Fehl-Scans zu vermeiden.

4. Variablen & Platzhalter

Damit das System weiß, welche Daten aus der CSV an welche Stelle auf dem Etikett sollen, nutzen wir Platzhalter.

Ein Platzhalter wird immer in [~ ~] eingeschlossen, zum Beispiel:

[~Materialnummer~]

Sie müssen diese nicht auswendig lernen! In den Eigenschaften eines Objekts (Doppelklick) finden Sie eine Liste aller verfügbaren Felder. Ein Klick darauf fügt den Platzhalter automatisch in den Text ein.

5. Etiketten-Formate & Druckeinstellungen

In der linken Seitenleiste stellen Sie das physische Papierformat ein:

Bogen-Layouts (z.B. Zweckform)

Geben Sie die Anzahl der Spalten und Zeilen pro Blatt an. Die Druckvorschau lässt Sie dann wählen, an welcher Stelle auf dem Blatt der Druck beginnen soll.

Rollen-Layouts (z.B. Brother)

Stellen Sie Spalten: 1 und Zeilen: 1 ein. Das System erkennt automatisch, dass es sich um eine Endlosrolle handelt und optimiert die Ansicht.

Drucker-Einstellungen am PC

Im Windows-Druckdialog des Browsers (Chrome/Edge) stellen Sie bitte "Ränder: Keine" ein und achten Sie darauf, dass die Skalierung auf "100%" (nicht "Anpassen") steht, damit die mm-Angaben exakt stimmen.

6. FAQ & Tipps

Das System nutzt eine High-Performance Filter-Engine. Bei Listen mit über 13.000 Zeilen reagiert die Suche verzögerungsfrei, da sie erst 0,3 Sekunden nach Ihrer letzten Eingabe startet ("Debouncing"). Das schont Ihren Prozessor und verhindert das Einfrieren der Seite.

Direkte .xlsx Dateien werden nicht unterstützt. Bitte speichern Sie Ihre Excel-Tabelle in Excel über "Speichern unter" als "CSV (Trennzeichen-getrennt) (*.csv)" ab. Diese Datei erkennt das System sofort und verarbeitet sie blitzschnell.

Dies ist ein Sicherheitsfeature. Die Artikeldaten liegen nur im flüchtigen Arbeitsspeicher (Session) Ihres Browsers. Sobald Sie den Browser schließen oder die Sitzung abläuft, werden diese Daten gelöscht. Ihr Etiketten-Design und die Projekteinstellungen hingegen bleiben sicher in der Datenbank gespeichert.

Das System ist lernfähig. Wenn Sie über "Reload CSV" eine Datei mit neuen Spalten hochladen, erkennt das System diese sofort und bietet sie im Designer als neue Platzhalter an. Bestehende Designs bleiben davon unberührt, es sei denn, eine benötigte Spalte wurde komplett gelöscht.

Technisch gibt es keine harte Grenze, wir empfehlen jedoch Listen bis ca. 20.000 Zeilen. Bei größeren Mengen kann die Anzeige im Browser langsamer werden. Für den Druck spielt die Menge keine Rolle, da das System das PDF effizient in Schüben generiert.

1. Steht die Skalierung im Druckmenü auf "100%"? Wenn "Anpassen" gewählt ist, werden die Striche minimal verzerrt.
2. Ist der Barcode zu klein? Vergrößern Sie das Objekt im Designer per Doppelklick und erhöhen Sie ggf. die Breite der Box.
